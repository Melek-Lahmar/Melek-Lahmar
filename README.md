## 📌 Featured Projects

### 1. E-commerce & Logistics Platform Integrated with Sage X3

Final-year project developed at **InfoSoft**.  
This project consists of an integrated e-commerce and logistics platform synchronized with **Sage X3**, designed to connect customer workflows, business operations, logistics tracking and ERP data.

The solution combines:

- A **React web application** for customers, sellers, supervisors and administrators
- An **ASP.NET Core Web API** as the central backend layer
- A **SQL Server database** for business data and system persistence
- A **Flutter mobile application** for logistics and delivery workflows
- **Sage X3 synchronization** for articles, catalogs, warehouses and stock data
- **n8n workflow automation** for chatbot and operational assistance

Main modules:

- Authentication and role-based access control
- Product catalog and search
- B2C / B2B customer workflows
- Cart, orders and delivery notes
- Stock and warehouse synchronization
- Delivery tracking and logistics workflows
- Complaints management
- Dashboards and reporting
- Mobile delivery features
- Workflow automation and chatbot support

**Tech stack:**  
ASP.NET Core Web API, .NET 8, C#, SQL Server, Entity Framework Core, React, TypeScript, Flutter, Sage X3, SignalR, Konnect, Brevo, n8n, Swagger, GitHub, Scrum.

Repository: [PROJET-PFE](https://github.com/Melek-Lahmar/PROJET-PFE)
