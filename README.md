## 📌 Featured Projects

### 1. E-commerce & Logistics Platform Integrated with Sage X3

Final-year project developed at **InfoSoft**.

This project consists of an integrated e-commerce and logistics platform synchronized with **Sage X3**, designed to connect customer workflows, business operations, logistics tracking and ERP data in a unified system.

The project was built as a multi-component solution combining web, backend, mobile, database and automation layers.

The solution includes:

- A **React web application** for customers, sellers, confirmers, supervisors and administrators
- An **ASP.NET Core Web API** as the central backend layer
- A **SQL Server database** for business data and system persistence
- A **Flutter mobile application** for logistics and delivery workflows
- **Sage X3 synchronization** for articles, catalogs, warehouses and stock data
- **n8n workflow automation** for chatbot and operational assistance
- **SignalR** for real-time communication and operational updates
- **Konnect** integration for payment workflows
- **Brevo** integration for transactional emails and notifications
- **Swagger** documentation for API testing and endpoint visualization

Main modules:

- Authentication and role-based access control
- Customer account creation and profile management
- Product catalog, search and filtering
- Product details, favorites and cart management
- B2C and B2B customer workflows
- Orders, quotations and delivery notes
- Stock and warehouse synchronization
- Sage X3 article, catalog, warehouse and stock synchronization
- Seller workspace and business order creation
- Confirmer workspace and order validation
- Delivery tracking and logistics workflows
- Cash-on-delivery workflow support
- Complaints management
- Inter-warehouse transit workflows
- Supervisor dashboard
- Administration dashboard
- Sales and logistics reporting
- Excel and PDF export features
- Flutter mobile delivery features
- n8n workflow automation
- Administrator chatbot support

**Tech stack:**  
ASP.NET Core Web API, .NET 8, C#, SQL Server, Entity Framework Core, React, TypeScript, JavaScript, Vite, Tailwind CSS, React Router, Axios, Zustand, Flutter, Dart, Provider, Sage X3, SignalR, Konnect, Brevo, n8n, Swagger, JWT, GitHub, Scrum, UML.
